# backend

### 중고 직거래 플랫폼

#### 주요 기능
1. 회원 기능 (회원가입, 로그인, 로그아웃, 회원탈퇴, 마이페이지)
2. 검색 기능 (필터링, 정렬)
3. 랭킹 기능 (이 달의 경매, 입찰이 많은 경매, 최신 등록 경매)
4. 경매 기능 (제품 등록/수정/삭제, 상세 페이지)
5. 결제 기능
